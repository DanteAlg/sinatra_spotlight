# Sinatra Spotlight

Turning the spotlight on the old Sinatra!

## Dependencies

* Ruby
* Sinatra
* Rack

## Install

`gem install sinatra_spotlight`

## sinspot commands!

- `sinspot` acts as `sinspot -v`, show gem version
- `sinspot new app_name` will create a sinatra app

## Run application

- `bundle`
- `rackup config.ru`
